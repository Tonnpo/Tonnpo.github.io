Simulating Geometric Brownian Motion¶

Recall a geometric Brownian motion is $$dS(t) = \alpha (t) S(t) dt + \sigma(t) S(t) dW(t)$$ where $W(t)$ is a Brownian motion. Or, equivalently

$$S(t) = S(0) \exp \left(\int_0^t(\alpha(s) - \frac{\sigma(s)^2}{2}) ds + \int_0^t \sigma(s) dW(s) \right)$$

For simplicity, let's assume that $\alpha(s) = \alpha, \sigma(s) = \sigma$ are constants

$$S(t) = S(0) \exp{ \left( (\alpha - \frac{\sigma^2}{2} )t + \sigma W(t)\right)}$$

To simulate geometric Brownian motions, we have two options:

  1. Discretize the Stochastic Differential Equation
  2. Use the analytic solution

Discretizing SDE¶

Assume $\alpha(t) = \alpha$ and $\sigma(t) = \sigma$ are constants. Then, $$dS(t) = \alpha S(t) dt + \sigma S(t) dW(t)$$ can be discretized as $$S_{t+\Delta t} - S_t = \alpha S(t) \Delta t + \sigma S(t) \left[W(\Delta t) - W(t)\right]$$ Note that $W(t+\Delta t) - W(t)= W(\Delta t)$ $$S_{t+\Delta t} = S_t + \alpha S(t) \Delta t + \sigma S(t) W(\Delta t)$$

Example¶

In [1]:
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

dt = 0.01
T = np.arange(0, 100, dt)

W_dt = np.random.normal(0, np.sqrt(dt), len(T)-1)
sigma = 0.2
alpha = 0.05

S = np.zeros(len(T))
S[0] = 100

for t in range(1, len(T)):
    S[t] = S[t-1] + (alpha * S[t-1] * dt) + (sigma * S[t-1] * W_dt[t-1])

plt.plot(range(len(T)), S)

Analytic Solution¶

Assume $\alpha(t) = \alpha$ and $\sigma(t) = \sigma$ are constants. Then, we can simulate $$S(t+\Delta t) = S(t) \exp \left((\alpha - \frac{\sigma^2}{2}) \Delta t + \sigma W(\Delta t) \right)$$

Example¶

import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

dt = 0.01
T = np.arange(0, 100, dt)

W_dt = np.random.normal(0, np.sqrt(dt), len(T)-1)
sigma = 0.2
alpha = 0.05

S = np.zeros(len(T))
S[0] = 100

for t in range(1, len(T)):
    S[t] = S[t-1] * np.exp((alpha - 0.5*(sigma**2)) * dt + sigma  * W_dt[t-1])

plt.plot(range(len(T)), S)

Which method?¶

Note that under mild conditions like in the example, both discretization and analytic solution are close.

In [3]:
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

dt = 0.01
T = np.arange(0, 100, dt)

W_dt = np.random.normal(0, np.sqrt(dt), len(T)-1)
sigma = 0.2
alpha = 0.05

S_a = np.zeros(len(T))
S_a[0] = 100

S_d = np.zeros(len(T))
S_d[0] = 100


for t in range(1, len(T)):
    S_a[t] = S_a[t-1] * np.exp((alpha - 0.5*(sigma**2)) * dt + sigma  * W_dt[t-1])
    
    S_d[t] = S_d[t-1] + (alpha * S_d[t-1] * dt) + (sigma * S_d[t-1] * W_dt[t-1])

plt.plot(range(len(T)), S_a, label='Analytic')
plt.plot(range(len(T)), S_d, label='Discretization')
plt.legend()

When do things go wrong?¶

Step size is too large¶

When the step size $\Delta t$ is large, the discrete approximation diverges from the analytic solution

Volatility is too high¶

If the step size being large is not bad enough, high volatility can cause it to become negative!
